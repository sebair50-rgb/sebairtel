# **App Name**: SebairTel AI Communicator

## Core Features:

- AI Smart Reply: Smart Reply Suggestions: Suggests contextually relevant replies based on received messages, enhancing user convenience and engagement in conversations. An AI tool quickly decides whether or not to make its suggestions contextually based on the content.
- Real-time Chat: Real-time Chat Interface: A dynamic messaging interface to facilitate seamless and interactive communication between users, complete with message statuses (sent, delivered, seen).
- Media Sharing: Media Sharing: Capability to share various media files (images, videos, and documents) directly within chats. Support all available <lucide-react> icons.
- Code Rendering: Code Block Rendering: Render code blocks within messages with syntax highlighting and execution capability.
- Profile Sidebar: Profile Sidebar: Allow opening a profile sidebar for the chat user.
- Personalized Settings: Settings Customization: Personalize user experience with settings for notifications, privacy, and dark/light mode themes.

## Style Guidelines:

- Primary color: #793EF6 (RGB), a vibrant violet to convey innovation and intelligence. 
- Background color: #18171F (RGB), a very dark desaturated violet for a clean, modern aesthetic. The use of dark mode reflects consideration of accessibility and reduces eye strain, particularly helpful for an application that is likely to be used often.
- Accent color: #39F67A (RGB), a bright, contrasting green, for call-to-action elements, highlights, and active states. As an analogous hue to the primary color, the accent enhances engagement while maintaining balance. Its high brightness and saturation also suit dark-mode applications by helping attract attention.
- Body and headline font: 'Inter', a grotesque-style sans-serif, gives a modern and neutral appearance.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use <lucide-react> icons for a consistent and modern look across the application.
- The interface will adopt a modern, minimalist layout, optimized for both desktop and mobile screens.
